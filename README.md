# 적금 다마고치 (Saviing)

> 적금을 넣을 때마다 성장하는 나만의 다마고치  
> 게임화를 통해 저축 습관을 만드는 핀테크 앱

<br/>

## 프로젝트 개요

- **개발 기간**: 2025.08 - 2025.09
- **팀 구성**: 6인 (Frontend 3, Backend 3)
- **수상**: SSAFY 특화 프로젝트 3등

### 1. 다마고치 육성 시스템
- 적금 납입 시 경험치 획득 및 레벨업
- 레벨에 따른 상호작용 기능 해금

### 2. 적금 관리
- 자유적금/정기적금 계좌 생성 및 관리
- 실시간 잔액 조회 및 거래 내역
- 목표 금액 설정 및 달성률 확인

### 3. 게임 요소
- 방 꾸미기 (3구역 커스터마이징)
- 다마고치 의상/액세서리 착용
- 게임머니 획득 및 아이템 구매

<br/>

## 기술 스택

### Frontend
- **Core**: React 18, TypeScript
- **상태 관리**: Zustand, TanStack Query
- **스타일링**: Tailwind CSS
- **라우팅**: React Router
- **빌드**: Vite

### Backend
- **Framework**: Spring Boot 3.x
- **Architecture**: Domain-Driven Design (DDD)
- **Database**: MySQL
- **Auth**: OAuth 2.0
